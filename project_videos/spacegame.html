Game Developer - Space Game Project

Project summary

I designed and developed a captivating 2D video game using Processing, showcasing my proficiency in both object-oriented programming and game development principles. The game features a straightforward yet engaging objective: players must navigate through progressively challenging levels, skillfully avoiding obstacles and enemies.

Key achievement:

Objective-driven Gameplay: The game immerses players in a dynamic experience where the primary goal is to navigate through a series of levels. Each level presents unique challenges, fostering an environment of continuous engagement and skill improvement.

Gradually Increasing Difficulty: The game design incorporates a thoughtful progression of difficulty across levels. This strategic approach ensures that players are consistently presented with new challenges, promoting skill development and maintaining an exciting gaming experience.
